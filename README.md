我们需要完成什么
1. 使用Android/iOS实现一个网页浏览器App
- 要求：该网页浏览器App需要实现如导航功能、历史记录查看、书签管理管理等基本功能，除此之外，还有一些可选的进阶需求功能开发，包括用户管理、JS Bridge、请求拦截、资讯阅读及网页图片浏览器等功能。
2. 分组：Android和IOS各一半，4-5人一组完成分组，每个同学只要求完成其中一个平台的App
